# Public Assets

## Imágenes Requeridas

Coloca las siguientes imágenes en esta carpeta:

### hero-image.png
- Dimensiones recomendadas: 1200x800px
- Descripción: Imagen de familia feliz para la sección hero
- Formato: PNG o JPG
- Ubicación: `/public/hero-image.png`

## Nota
Si no tienes la imagen, la aplicación funcionará de todas formas. 
Solo se mostrará un área con color de fondo degradado.

Puedes descargar imágenes de bancos de imágenes gratuitos como:
- Unsplash
- Pexels
- Pixabay

Busca términos como "happy family", "health insurance", "familia feliz".
